# deckboss

> Deckboss.ai — AI-powered system design for edge robotics and IoT.

**Live:** [deckboss.casey-digennaro.workers.dev](https://deckboss.casey-digennaro.workers.dev)

Describe what you need. Get wiring diagrams, simulations, parts lists, and installation guides. Hand off to Cocapn when ready to deploy.

## Features

- **Visual System Design** — Describe your project, get wiring diagrams and 3D-ready specs
- **Photo-to-Wiring** — Photograph existing hardware, get connection diagrams
- **IoT Architecture** — Sensor networks, compute topologies, firmware suggestions
- **Education On Board** — Token-based course generation, shared content is free
- **Hand Off to Cocapn** — Design becomes the running system on hardware
- **TUI in Every Repo** — Opens in terminal or browser, no installation

## Content Economy

- Generate fresh content with tokens
- Share freely with other users (shared content = free)
- Remix popular content for new environments
- Community flywheel: more users = more shared content = lower costs for everyone
